## Getting Started

Follow these steps to get started with this backend

### SetUp BE
```bash
npm install
``` 
```bash
npx sequelize-cli db:migrate
``` 
```bash
npx sequelize-cli db:seed:all
``` 

### Running a Development Server

Start the development server:

```bash
npm run start
```
